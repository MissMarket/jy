# 前端设计

该技能帮助创建视觉吸引力强且可直接用于生产的前端设计。它提供有关网页应用布局、配色方案、排版和整体视觉美学的指导。

## 使用场景

- 创建新的前端界面时
- 改进现有 UI 设计时
- 寻求网页应用视觉增强时
- 设计响应式布局时
- 需要可直接用于生产的设计解决方案时

## 核心功能

- 高质量的设计建议
- 独特且视觉吸引力强的布局
- 可直接用于生产的实施指南
- 响应式设计考虑
- 配色方案和排版最佳实践

## 使用示例

- "为我们的分析工具设计现代仪表板界面"
- "提升我们产品着陆页的视觉吸引力"
- "为我们的网站创建响应式导航菜单"
- "增强我们移动应用的 UI"

## 输出格式

该技能提供设计建议、代码片段和实施指南，以实现符合生产标准的高质量前端设计。
